É possivel posicionar elementos não texto também. Desde que esse elemento possua algum tamanho de referência.

Baste que adicione o alinhamento no lugar que vai receber o(s) elemento(s) para alinhar.

Então exibimos eles como elementos em linha:

<div class="bg-black box-x-auto align-center">
    <div class="box-xy-200 bg-red margin-all line-block"></div>
    <div class="box-xy-200 bg-red margin-all line-block"></div>
</div>

Alinhamento vertical

Nota* - Para que o alinhamento vertical funcione é necessário tratar-lo como uma tabela.

Quando os elementos internos que serão alinhados, não possuírem um tamanho de altura é necessário que então o elemento pai tenha uma altura referencial. Caso contrário todos os elementos internos serão exibidos como base da linha de conteúdo existente.

vertical-baseline
vertical-bottom
vertical-middle
vertical-top 
<div class="bg-black box-fluid-99 box-y-250 align-center table">
    <div class="vertical-baseline box-fluid-24d border-all border-white">vertical-baseline</div>
    <div class="vertical-bottom box-fluid-24d border-all border-white">vertical-bottom</div>
    <div class="vertical-middle box-fluid-24d border-all border-white">vertical-middle</div>
    <div class="vertical-top box-fluid-24d border-all border-white">vertical-top</div>
</div>

Alinhamento Horizontal & Vertical

Esse método alinha um elemento ao centro de elemento pai na horizontal e vertical.
É necessário que o elemento pai possua uma altura de referência.
SÓ FUNCIONA EM NAVEGADORES DE TOTAL COMPATIBILIDADE COM HTML5! (Chorme/Mozila/Opera/Safari)
vertical-wrap 
<div class="bg-black box-fluid-99 box-y-400 vertical-wrap">
    <span class="font-medium text-white">vertical-wrap</span>
</div>
