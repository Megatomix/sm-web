Exibição

Métodos de manpulação de como elementos devem ser exibidos.

block

Este valor faz com que um elemento gere uma caixa de bloco.

none

Esse valor faz com que um elemento não gere caixas na estrutura de formatação (ou seja, o elemento não tem efeito no layout).

Elementos descendentes também não geram caixas; esse comportamento não pode ser substituído definindo a propriedade 'display' nos descendentes.

Note que uma exibição de 'none' não cria uma caixa invisível; Não cria nenhuma caixa.

O CSS inclui mecanismos que permitem que um elemento gere caixas na estrutura de formatação que afetam a formatação, mas que não são visíveis.

list

Esse valor faz com que um elemento (por exemplo, LI em HTML) gere uma caixa de bloco principal e uma caixa sequencial de item de lista.

Exemplo:

a

b

c

<p class="list">a</p>
<p class="list">b</p>
<p class="list">c</p>

table

Esse valor faz com que uma caixa seja exibida como uma tabela HTML.

trow

Esse valor faz com que uma caixa seja exibida como uma linha de tabela.

thead

Esse valor faz com que uma caixa seja exibida como um cabeçalho de tabela.

tbody

Esse valor faz com que uma caixa seja exibida como estrutura de uma tabela.

tcol

Esse valor faz com que uma caixa seja exibida como coluna de uma tabela.

tgroup

Esse valor faz com que uma caixa seja exibida como pertecente ao grupo de uma tabela.

tcell

Esse valor faz com que uma caixa seja exibida como parte de uma tabela.

tcap

Esse valor faz com que uma caixa seja exibida como conteúdo de tabela.
NOTA * Todas as regras para tipo tabela nada mais é que:

Fazer com que o CSS interprete outros elementos que não sejam tabelas ou seu conteúdo como se fossem tal.

Para mais informações sobre o que é uma tabela e seu comportamento no HTML.

Leia essa documentação: www.w3.org/TR/CSS2/tables.html

line-block

Exibe uma caixa como bloco de linha.

line-table

Exibe uma caixa como bloco de tabela.
Possuíndo essas regras junto a conhecimento e experiência combinado com outras regras é possível elaborar layouts com certa característica única.

Confira abaixo alguns exemplos:

Posição central na vertical

<div class="table bg-black box-fluid-99 box-y-200">
    <div class="tcell vertical-middle">
        <p class="font-default text-white">Posição central na vertical</p>
    </div>
</div>

Posição central na vertical e horizontal

<div class="table bg-black box-fluid-99 box-y-200">
    <div class="tcell vertical-middle align-center">
        <p class="font-default text-white">Posição central na vertical</p>
    </div>
</div>
<div class="margin-top-high">
    <div class="box-fluid-33b box-y-200 bg-red line-block" style="margin-left: -5px"></div>
    <div class="box-fluid-33b box-y-200 bg-green line-block" style="margin-left: -5px"></div>
    <div class="box-fluid-33b box-y-200 bg-blue line-block" style="margin-left: -5px"></div>
</div>
As propiedades display-*? ou table-*? não afetam as propiedades float.
