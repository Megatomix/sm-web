Quebra palavra

Usada para quebrar uma sequência de texto lado a lado.

Quando um texto (sem uso de espaços) for maior que o elemento pai, o navegador pode fazer com que esse texto vaze ao elemento que se encontra. Para previnir isso e ser for realmente um problema, essa regra é a solução.

Basta que adicione o método break ao texto.

Forçando a maximização

É posssível forçar um elemento assumirem total extensão, vejamos os métodos.
Redimencione a tela para visualizar os efeitos
Primeiro vamos ver um boco padrão sem aplicar a ampliação.

Indepedente da resolução ele sempre vai se manter do mesmo tamanho.

box-xy-150

Agora aplicamos o método maximized ao qual fará com que esse bloco sempre tenha total extensão.

box-xy-150 
<div class="box-xy-150 maximized bg-red table align-center">
    <div class="font-medium text-white table-cell vertical-middle">
