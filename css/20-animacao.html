Efeitos

Com o CSS3 veio uma grande facilidade para o desenvolvimento, inclusive o uso de efeitos.

Antigamente para se criar um efeito sempre necessitava-mos de muitas linhas de código aos quais nem sempre saíam como esperado.

Então não pude deixar de abordar alguns efeitos e definir-los no arquivo CSS padrão.

Aqui vemos esses efeitos

Clique no botão para exibir o efeito (*! Todos efeitos aqui são acionados por javascript) 





<button style="min-width: 230px" class="btn-default margin-all">spinner</button>
<button style="min-width: 230px" class="btn-default margin-all">spinner-off</button>

<button style="min-width: 230px" class="btn-default margin-all">rotate</button>
<button style="min-width: 230px" class="btn-default margin-all">rotate-off</button>

<button style="min-width: 230px" class="btn-default margin-all">turn</button>

Ver javacript para ativação dos efeitos

var btn = document.getElementsByClassName('btn-default');
var time;
for (var i = 0; i < btn.length; i++) {
    btn[i].addEventListener('click', function () {
        var txt = this.innerText;
        this.classList.toggle(txt);
        time = setTimeout(function () {
            removeEffcts();
        }, 2000);
    });
}

function removeEffcts() {
    clearTimeout(time);
    for (var i = 0; i < btn.length; i++) {
        btn[i].classList = 'btn-default margin-all';
    }
}
