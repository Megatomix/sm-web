Em alguns casos nosso elemento absoluto, ele pode ficar oculto atrás de outro elemento.

Nesse caso passamos a sua importancia ser maior que o elemento á frente.

Para isso usamos a propiedade z-index

Exemplo: 
<ul class="list-none absolute none bg-dark hover-show-item" style="z-index: 1">
z-index definine o grau de importância de um elemento, quanto maior seu valor, maior será sua importancia de exibição. Essa propiedade afeta drasticamente qualquer elemento cujo possua a propiedade position.

Resposivo Ativo

Para esse tipo de menu devemos utilizar o javascript para exibir o elemento.

É necessário que a resolução seja inferior a 600 pixel para visualização desse efeito.

Usaremos vários métodos do nosso CSS padrão para ocultar, reinscalar e posicionar elementos.

Aqui optei em usar as regras com a resolução de 600px.

Assim sendo quando a resolução for inferior a 600 exibimos o item que abrirá nosso menu responsivo
