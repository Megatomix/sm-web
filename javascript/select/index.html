SelectOption.js

Desde sempre a personalização do de um grupo de opções foi obstáculo para se fazer utilizando simplesmente o CSS.

E é fato que cada navegador o interpreta diferente, então muitas vezes ao ser necessário a utilização, o mesmo pode fazer com que seu disign não combine com o restante da estrutura do HTML na página.

Pensando nessa situação desenvolvi esse javascript para personalizar essa quase que indispensável aplicação de desenvolvimento.

Select padrão do navegador

<select style="width: 100%" class="font-medium">
    <option value="a">Selecione uma opção</option>
    <option value="b">Opção</option>
    <option value="c">Opção</option>
    <option value="d">Opção</option>
    <option value="e">Opção</option>
</select>

Select personalizado pelo javascript

<select class="select">
    <option value="a">Selecione uma opção</option>
    <option value="b">Opção</option>
    <option value="c">Opção</option>
    <option value="d">Opção</option>
    <option value="e">Opção</option>
</select>
Para a implementação faça o download dos arquivos necessários:
Uma vez que tenhamos os arquivos inserimos eles dentro da tag head no index: 
<script src="SelectOption.mim.js" type="text/javascript"></script>
<link href="SelectOption.mim.css" rel="stylesheet" type="text/css"/>
Pós o procedimento de implementação temos que iniciar os métodos do javascript.

Para vamos ao final do documento antes do fechamento da tag body e criamos um objeto para declarar o script: 

<!-- Restante do código HTML acima -->
        <script>var objSelect = new SelectOption();</script>
    </body>
</html>
Pronto com isso para que cada grupo de opções seja personalizado, basta que o mesmo possua uma class selected 
<div class="row-pad font-medium">
    <div class="col-third">
        <select class="selected">
            <option value="option_A-1">Opção A-1</option>
            <option value="option_A-2">Opção A-2</option>
            <option value="option_A-3">Opção A-3</option>
        </select>
    </div>
    <div class="col-third">
        <select class="selected">
            <option value="option_B-1">Opção B-1</option>
            <option value="option_B-2">Opção B-2</option>
            <option value="option_B-3">Opção B-3</option>
        </select>
    </div>
    <div class="col-third">
        <select class="selected">
            <option value="option_C-1">Opção C-1</option>
            <option value="option_C-2">Opção C-2</option>
            <option value="option_C-3">Opção C-3</option>
        </select>
    </div>
</div>
É importante que saiba, que todo texto de uma opção será reduzido caso o mesmo possua mais de 20(vinte) caracteres. Mas o value permanece inalterado.

Essa redução do texto base é por questões estéticas da personalização.

A utilização dessa aplicação é puramente por questões de estilizar um grupo seletor.

Então para que você possa estilizar a seu modo, você deve editar o CSS. Abaixo temos a estrutura de produção para melhor acesso as propiedades no ambiente de desenvolvedor:
